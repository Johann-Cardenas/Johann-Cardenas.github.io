How to Use

  1. Select Aircraft — Choose a manufacturer and model from the FAARFIELD library.
  2. Review Parameters — Check auto-populated load, pressure, and rib geometry. Adjust if needed.
  3. Compute — Click Compute (or Ctrl+Enter) to run the stress analysis.
  4. Explore Results — Use the tabs to visualize 3D surfaces, distributions, and export data.

Adjust Contact Length?

Changing the wheel load requires scaling the contact length to maintain physical consistency. Contact width is held constant (tire carcass constraint), while contact length scales linearly with load at fixed inflation pressure.
